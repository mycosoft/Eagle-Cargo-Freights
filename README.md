# Eagle Cargo Freights

**Logistics Management System**

Eagle Cargo Freights is a comprehensive logistics management platform for tracking air and sea cargo shipments, managing clients, handling invoices and payments, and facilitating internal team communication.

---

## Features

- **Air Cargo Management** - Full lifecycle tracking with dedicated dashboard
- **Sea Cargo Management** - Container tracking with CBM calculations and dedicated dashboard
- **Batch/Container Management** - Group shipments, track expenses, calculate profit/loss
- **Client Management** - Import clients, track shipment history
- **Invoicing & Payments** - Auto-generated invoices, payment recording with receipt PDFs
- **Expense Management** - Track operational expenses in UGX and USD
- **Internal Chat** - Real-time messaging between users with file/image attachments
- **Notification System** - Bell notifications for approvals, rejections, and system events
- **Activity Logs** - Full audit trail tracking who did what
- **Role-Based Access** - Granular permissions using Spatie Laravel Permission
- **WhatsApp Integration** - Send shipment status updates via WhatsApp
- **Comprehensive Reports** - Revenue, shipments, clients, outstanding balances

---

## Tech Stack

- **Backend:** Laravel 12
- **Frontend:** AdminLTE, Bootstrap, Alpine.js, Chart.js
- **Database:** MySQL
- **Build:** Vite
